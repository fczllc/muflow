Mermaid流程图测试

本页面包含多个Mermaid流程图示例，可用于测试导入功能。点击"复制"按钮将示例复制到剪贴板，然后在MuFlow应用中粘贴。

示例1：基本流程图

flowchart LR
    A[开始] --> B{是否满足条件?}
    B -->|是| C(处理数据)
    B -->|否| D(跳过处理)
    C --> E[结束]
    D --> E

示例2：复杂流程图

flowchart TD
    A[开始] --> B[初始化]
    B --> C{检查数据}
    C -->|有效| D[处理数据]
    C -->|无效| E[报错]
    D --> F[保存结果]
    E --> G[记录日志]
    F --> H[结束]
    G --> H

示例3：左右布局流程图

flowchart LR
    A([开始]) --> B(准备数据)
    B --> C{数据验证}
    C -->|成功| D((处理))
    C -->|失败| E[清理]
    D --> F[保存]
    E --> G([结束])
    F --> G
